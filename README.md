■基本操作

[キーボード]<br>
Z：通常弾発射　　<br>
X：ボム発射　　<br>
Q：ポーズ　　<br>
Shiftキー: 低速移動モード　　<br>
方向キー：キャラ移動、選択カーソル移動　　<br>
Enterキー: 決定、会話パート進める　　<br>

■カリスマポイントについて

敵を倒すもしくはボムで敵弾を相殺するとカリスマポイントが発生します<br>
カリスマポイントは...<br>
 ・敵を倒す→100ポイント<br>
 ・ボムで敵弾を相殺する→10ポイント<br>
取得できます<br>
5000ポイントごとにボムを1つ、10000ポイントごとに自機を1つ獲得できます<br>
またポイント一定量貯まるとレミリアの姿が変わり、通常弾とボムが進化します<br>
